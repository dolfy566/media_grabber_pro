 MediaGrabber Pro

MediaGrabber Pro es una herramienta multiplataforma para descargar videos y audios desde múltiples sitios web como YouTube, TikTok, Facebook, Twitter, entre otros.

Compatible con:
- ✅ Windows (versión con interfaz gráfica)
- ✅ Linux (con instalación automática de dependencias)

---

## ✨ Características principales

- Descarga de videos individuales o listas de reproducción
- Conversión automática a MP3
- Selección de calidad (video y audio)
- Interfaz gráfica multilingüe (Español, Inglés, Portugués BR, Alemán, Francés)
- Modo oscuro, historial de descargas, logs y más
- Detección automática de dependencias (`yt-dlp`, `ffmpeg`) en Linux
- Dos modos de descarga: video único o playlist
- Barra de progreso individual por descarga

---

## 🖥️ Instalación en Linux

```bash
chmod +x install.sh
./install.sh
```

Esto instalará automáticamente las dependencias necesarias (`yt-dlp`, `ffmpeg`, `tkinter`, etc.) y creará accesos directos en el sistema.

---

## 🪟 Uso en Windows

Ejecuta directamente `MediaGrabberPro.exe`.  
No requiere instalación previa ni reproductores adicionales.
