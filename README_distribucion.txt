MediaGrabber Pro - Distribución y Licencias

Este paquete contiene el ejecutable media_grabber_pro_win.exe generado a partir del código fuente en Python.

Instrucciones de uso:
- El archivo ejecutable se encuentra en la carpeta 'dist' dentro del directorio principal.
- Simplemente ejecute media_grabber_pro_win.exe para iniciar la aplicación.
- No es necesario tener Python instalado para usar el ejecutable.
